# Задание 2.2: Тесты UI

## Подготовка системы

```bash
# 1. Клонировать репозиторий
git clone https://github.com/n1ck-L/avito-qa-internship-2026
cd avito-qa-internship-2026

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Установить Chromium для Playwright
playwright install chromium
```

## Запуск тестов

К сожалению доступ к сайту https://cerulean-praline-8e5aa6.netlify.app/ с заданием невозможен без настроенного прокси-сервера или включенного VPN. По этой причине ниже описаны все способы для запуска автотестов для изежания данной проблемы.

#### Режим 1. Без прокси

Тесты идут напрямую через интернет-соединение и запущенный VPN.
```bash
python main.py

```
#### Режим 2. C прокси

Указываете работающий прокси-сервер вручную в формате IP:PORT.
```bash
python main.py --proxy 95.213.217.168:52004
```
#### Режим 3. Автоматический поиск прокси

Система сама пытается найти рабочий прокси. Есть шанс, что он не будет найден
```bash
python main.py --proxy auto
```
